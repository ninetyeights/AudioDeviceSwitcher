# 更新日志

## 1.1.0 — 2026-04-26

### 新增

- **默认设备 peak 电平条**：主窗口默认播放 / 录音设备显示横向实时电平（NAudio `AudioMeterInformation`）；录音设备用 `WasapiCapture` keepalive，不再依赖系统录音界面打开
- **Voicemeeter 集成增强**：
  - Strip / Bus 实时 peak 电平条（`VBVMR_GetLevel`，零额外开销）
  - Strip 静音状态显示 + 一键切换（点击徽章），写入即时反馈
  - 设备丢失检测：Voicemeeter 配置的物理设备从系统消失时显示红色"已丢失"标签
  - Strip 静音状态锁定：外部从 Voicemeeter UI 修改会自动恢复，后台线程 50ms 紧轮询
- **配置锁定**：
  - 每个 profile 独立 🔒 / 🔓 切换按钮
  - 锁定后系统面板切换设备时立即恢复（COM 事件直驱，无 debounce）
  - 锁定时软件内点其他 profile / 设备右键"设为默认 / 启用 / 禁用"全部禁用 + 提示
- **Voicemeeter 集成开关**：设置里加"启用 Voicemeeter 集成"复选框（默认关闭）；关闭时不加载 DLL、不轮询，无开销
- **配置拖拽排序**：profile 卡片左侧 ☰ 句柄拖动调整顺序
- **配置标签颜色**：8 色预设（蓝/绿/红/黄/紫/粉/灰/无），主窗口和迷你窗口都显示色点
- **配置"在迷你窗口中显示"开关**

### 优化

- **主窗口布局**：播放 / 录音设备改为左右两栏，Voicemeeter 面板与配置区并排，整体紧凑
- **静音切换响应**：乐观更新 + pending writes 表，连续点击不会被异步刷新覆盖
- **后台轮询**：Voicemeeter 改为后台线程，锁定时 50ms 紧轮询，未锁定 1.5s

### 修复

- 主窗口最小化时正确停止 peak 计量，麦克风占用图标消失
- Voicemeeter 标题与"配置"标题视觉对齐
